## Dev guide

Run a devnet:

```shell
yarn devnet
```

Deploy contracts locally:

```shell
yarn deploy:local
```

Run tests:

```shell
yarn test
```

Run tests with coverage:

```shell
yarn coverage
```

Deploy new version:

```shell
yarn publish-to-npm
```
